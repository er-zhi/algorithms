# Algorithms

Popular algorithms with **theory** (markdown) and **code** (JS). Each folder: `README.md` + LC-numbered solution files.

## Contents

| Folder | Description | Theory |
|--------|-------------|--------|
| [Kadane-Algorithm](Kadane-Algorithm/) | Max sum of contiguous subarray (LC 53) | [README](Kadane-Algorithm/README.md) |
| [Divide-and-Conquer](Divide-and-Conquer/) | Merge sort, D&C pattern (LC 912, etc.) | [README](Divide-and-Conquer/README.md) |

## Structure

```
<Algorithm-Name>/
  README.md
  <number>-<slug>.js  # e.g. 53-maximum-subarray.js
```

Run from repo root (no spaces in paths):

```bash
node Kadane-Algorithm/53-maximum-subarray.js
node Divide-and-Conquer/912-sort-an-array.js
```
